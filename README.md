Папка APK имеет APK установщик мобильного приложения.
Папка InvestMobile имеет структуру мобилного приложения.
Папка InvestBuy имеет структура сайта.

1.1 Что нужно установить
Скачайте и установите Node.js	
Откройте командную строку (cmd) или PowerShell и выполните:
node --version
Если видите версию (например v18.17.0)  всё готово

2: СТРУКТУРА ПРОЕКТА
2.1 Где находятся файлы
 backend                 
 frontend  
 
3: ЗАПУСК BACKEND (СЕРВЕРА)
3.1 Откройте первый терминал в папке backend

3.1 Установите зависимости
npm install
Эта комоанда скачает все необходимые библиотеки
Что должно появиться:
Папка node_modules
Файл package-lock.json

3.2 Запустите сервер
npm start
или для разработки (автоперезагрузка при изменениях)
npm run dev

ВАЖНО Этот терминал должен оставаться открытым всё время работы сайта

4: ЗАПУСК FRONTEND (САЙТА)

4.1 Откройте ВТОРОЙ терминал в папке Frontend

4.2 Установите зависимости
npm install

4.3 Запустите сайт
npm start

4.4 Что должно произойти
Автоматически откроется браузер с адресом:
http://localhost:3000
